# Backend

Parte backend do projeto, desenvolvida com _NodeJS_ e _Express_.

A conexão com ambos os ocorrem com o uso de APIs, utilizando o _Express_.

A conexão com o banco de dados é feita com _mysql2_.

## Desenvolvimento

É preciso ter o _NodeJS_ e _NPM_ instalados.

Instalação de dependências do projeto: `npm install`.

## Testes

Automáticos, utilizam o _Jest_ e _Supertest_.

Testes unitários: `npm test`.

## Execução

Iniciar o server: `npm start`.

É necessário que o database esteja sendo executado.
